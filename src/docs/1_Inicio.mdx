Dimacheris es un template para documentación construido con Astro y pensado para que no tengas que
preocuparte por la configuración.

## Instalación

Para instalar el template abre una terminal y escribe lo siguiente:

<Code
  lan="sh"
  code="npm create astro@latest -- --template Iannacus/dimacheris-template"
/>

Esto creará una carpeta con los archivos que necesitarás para iniciar con tu documentación.

<Alert type="alt" title="Atención">
  En el asistente de configuración es preferible que permitas la instalación de
  las dependencias
</Alert>

## Ejecuta tu proyecto

Para visualizar el contenido de lo que estas trabajando será necesario iniciar con un servidor local.

<Code lan="sh" code="npm run dev" />

Es necesario que en un navegador escibas la url que se muestra por la consola <br />
P.E. locahost:4321

<Alert type="warning">
  No existe, de momento, una página de inicio. Para acceder al contenido agrega:
  **/example** a la url.
</Alert>

Con esto ya tienes tu proyecto ejecutandose y listo para que agregues contenido.
